    PowerShell System Queries

    Find PC Specs

    Get-ComputerInfo

    Find GPU

    gwmi win32_VideoController | FL Name

    Find RAM

    Get-WmiObject Win32_PhysicalMemory |
Format-Table BankLabel, Capacity, Manufacturer

    Find CPU

    Get-CimInstance Win32_Processor

    Find Storage

    Get-Disk | Format-Table -AutoSize
